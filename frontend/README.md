# GOMOKU Frontend


## Installation


```bash
npm install
or
yarn
```


## Environment Variables

VITE_REACT_APP_API_URL: Local frontend address
VITE_NODE_APP_API_URL: Local backend address - pre-routes

## Usage

For development
```javascript
npm run dev
```
For deployment
```
npm run build
```
